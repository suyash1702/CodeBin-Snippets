This code demonstrates a deferred loading mechanism for an Angular component with the following key features:

  • Deferred Loading: The <app-heavy> component is loaded only when the user hovers over the "Hello!" button. This approach helps in improving performance by delaying the loading of resource-intensive components.
  • Placeholder Content: While waiting for the <app-heavy> component to load, a placeholder message ("this is place holder") is displayed to inform the user that content will appear soon.
  • Loading State: If the loading process takes more than 100 milliseconds, a "Content is loading" message is shown. This provides feedback to the user that the component is being fetched and helps manage user expectations.
  • Error Handling: If an error occurs while loading the <app-heavy> component, an error message ("Error while loading heavy component") is displayed. This helps in notifying the user about issues during the component loading process.
